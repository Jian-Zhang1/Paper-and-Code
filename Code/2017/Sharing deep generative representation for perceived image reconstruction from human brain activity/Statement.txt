This is the code for paper "Sharing deep generative representation for perceived image reconstruction from human brain activity". It contains the implementation of the Deep Generative Multi-view Model (DGMM) which was described in the paper.

The code was downloaded through github.com.

The paper must be cited as the following if you want to use this code in your own work.

@inproceedings{du2017sharing,
  title={Sharing deep generative representation for perceived image reconstruction from human brain activity},
  author={Du, Changde and Du, Changying and He, Huiguang},
  booktitle={IJCNN},
  year={2017}
}

You will find more information in README.md.

Here is the original link: https://github.com/ChangdeDu/DGMM/tree/684daf6eb531cfc812bfadf72e07563295264ea2.
